EcoDefense 
EcoDefense est un jeu de type Tower Defense où le but est d'éliminer des ennemis pour passer à la prochaine vague

Contraintes : 
Version de Python 3.8 ou supérieur
Version de Pygame 2.6.1 (voir requirements.txt) 

Le fichier main.py est le seul point d'entrée du projet et est le seul endroit
où le code doit être lancé. 
Installer Pygame avec la commande : "pip install pygame==2.6.1" dans le terminal
de l'éditeur de code
Avec n'importe quel éditeur de code lancer le projet en appuyant sur Run en étant 
dans le fichier main.py

Le jeu a été testé exclusivement sur le logiciel Visual Studio Code sur la 
version 1.110.1 ainsi sur qu'un appareil Windows avec les chemins d'accès utilisant "/".
